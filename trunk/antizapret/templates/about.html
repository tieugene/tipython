Система предназначена для работы с регламенированными документами - создание, изменение, печать.
Система разработана гибкой, с возможностью лёгкого подключения любого количества шаблонов.
Поддерживает печать как "мягких" документов (в достаточно свободной форме) - так и "жестких" (для сканирования, текст имеет жесткие координаты).

Работает в трех режимах:

Демо

для незарегистрированных пользователей - позволяет заполнить шаблон и тут же распечатать итоговый документ.

Стандартный

для зарегистрированных пользователей = Демо + сохранение и редактирование документов.

Расширенный

платный = Стандартный + дополнительные функции.

Во всех режимах:

Справочники общие

Общероссийские классификаторы, Банки, МИФНС, иное; заполнение форм из этих справочников.

Проверка полей

Проверка корректности заполнения таких данных, как ИНН, КПП, БИК, иное.

Заказные шаблоны

Создание новых шаблонов на заказ - уникальных для пользователя или общеупотребительных. Цена определяется отдельно для каждого шаблона. Можно даже оплатить вскладчину (если шаблон нужен более чем одному пользователю).

В Расширенном режиме:

Сохранить как

Копирование данных документа в новый документ (того же вида) - т.е. дублирование документов.

Заполнить на основании

Заполнение документа одного вида данными документа другого вида (где это возможно).

Справочники личные

Организации, Люди, Документы, Адреса; заполнение форм из этих справочников, сохранение данных форм в справочники.

Печать пачки документов

Печать нескольких сохраненных документов (любого типа) за раз.

Заполнение комплекта документов

Заполнение комплекта связанных документов в одной форме ввода.

Кроме того - для любителей поупражняться в HTML есть возможность заработать на создании новых шаблонов.

Термины:

Шаблон (вид документа) - форма ввода данных + шаблон печатной формы (бланк).

Документ - Шаблон, заполненный данными.
