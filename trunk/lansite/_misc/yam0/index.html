How to use the menu ?

It is tested in Firefox, IE6 and IE7, Opera and Safari

Search Engine friendly, it parses a raw ul and li structure (see below)

First download it.


Extract it, then in your html page, create a structure like this:

<!--START MENU-->
<ul id="Menu1" style="display:none;">
<li>An Option without link</li>
<li><a href="http://www.google.com/">An option with link</a>
<ul>
<li>A sub option</li>
</ul>
</li>
</ul>
<!--END MENU-->

Then the divs that holds the js menu:

<div align="center">
<div id="MenuContent" style="background-color:black; width:380px; height:27px;"></div>
</div>

Its important for browsers like IE that the div holding the menu, in this case "MenuContent", is not contained in styled divs, for example divs with style="width:200px". This is because in some browsers the method that reads the parent's position gets disaligned.
